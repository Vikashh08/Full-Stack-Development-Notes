# JavaScript Learning Guide

Welcome to the JavaScript learning repository. Here you can learn JavaScript concepts conceptually along with commonly asked interview questions.

## Topics Covered

This repository contains detailed explanations and interview questions on the following core JavaScript topics:

- **Definition**: Basic definitions and concepts in JavaScript.
- **Execution Context**: Understanding how JavaScript code evaluates and executes.
- **Call Stack**: How the JavaScript engine keeps track of function calls.
- **Hoisting**: How variable and function declarations are moved to the top of their scope.
- **Scope**: Lexical environment and understanding access to variables.

## How to use this guide

Each markdown document in this repository is designed to give you a thorough understanding of the topic, followed by relevant interview questions that will test your knowledge and help you prepare for technical interviews.

Happy learning.
